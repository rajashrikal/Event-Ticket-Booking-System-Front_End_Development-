# 🎟️ EventSpark – Event Ticket Booking System

A responsive front-end ticket booking website built using HTML, CSS and JavaScript.

## Features
- Browse events (movies, concerts, sports)
- Book tickets with form validation
- Booking confirmation with summary
- Fully responsive (mobile, tablet, desktop)

## Built With
- HTML5
- CSS3
- JavaScript

## Levels
- Level 1: Page Structure & Layout
- Level 2: Styling & Visual Design
- Level 3: Event Display & UI Components
- Level 4: Front-End Interaction
- Level 5: Responsive Design & Final Touches
